# controlled_square_via_arrows
### MIPS based programe to control a square motion by a,d,w,s,c leters (left,right,up,down,termination respectively).<br/>


**It has been implemented using ***MIPS*** assemply using ***MARS simulator***.** <br/><br/>

## Keypoints 
1. configs for Bitmap Display ( dimensions , units (squares) addresses ...) .
2. Data Segment 
3. Text Segment
4. Macros Definetion
5. Handling the pressed key via Receiving Buffer of the keyboard<br/><br/><br/>

## To run it 
```
1. git pull main your_local_branch
```
2. Open MARS , go to Tools bar then select the **Bitmap Display** & **keyboard and display MIMO simulator** .
3. Set up the **Bitmap Dispaly** configs as mentioned in the code .<br/><br/><br/>
